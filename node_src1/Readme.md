
## Install

Install dependencies

```shell
npm install express -g

npm install
```

## Config

See `config.js`.

The `dbPath` is the path witch Level DB folder.

## Command

Generate default record value:

```shell
npm run gen
```

Run reduce:

```shell
npm run reduce
```

Test api:

```shell
npm run api
```

Run server to see report chart:

```shell
npm run serve
```
